# User Guide – Getting Started with UniSync

## What is UniSync?
UniSync is an AI-powered assistant that helps answer questions related to University of Cincinnati academics and services by searching publicly available university information.

It is designed to save students time by providing quick, centralized answers instead of searching multiple websites.

## What Can You Ask UniSync?
You can ask UniSync questions such as:
- “When is the application deadline?”
- “How do I apply for scholarships at UC?”
- “Where can I find course requirements?”
- “How do I contact the registrar?”

UniSync works best with clear, specific questions written in natural language.

## How to Start Using UniSync
1. Open the UniSync web application.
2. Type your question into the input box (or use voice input if available).
3. Submit your question.
4. Review the generated response and any provided sources.

## Example Questions
- “What are the admission requirements for engineering programs?”
- “How do I declare a major at UC?”
- “Where can I find information about tuition and fees?”

## Intended Use
UniSync is meant to be a **support tool** that helps users quickly locate information. It should not be used as a replacement for official university offices or academic advisors.
